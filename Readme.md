AWS Academy Cloud Architecting [130452]

AWS Academy Cloud Architecting cubre los fundamentos de la creación de infraestructura de TI en Amazon Web Services o AWS. El curso está diseñado para enseñar a los arquitectos de soluciones cómo optimizar el uso de la nube de AWS mediante la comprensión de los servicios de AWS y cómo estos servicios se adaptan a las soluciones basadas en la nube.

Dado que las soluciones arquitectónicas pueden variar según la industria, el tipo de aplicaciones y el tamaño de la empresa, este curso hace hincapié en las prácticas recomendadas para la nube de AWS y recomienda varios patrones de diseño para ayudarle a pensar en el proceso de diseñar soluciones de TI óptimas en AWS. También presenta casos prácticos a lo largo del curso, que muestran cómo algunos clientes de AWS han diseñado sus infraestructuras, así como las estrategias y servicios que han implementado. Por último, este curso también ofrece oportunidades para construir una variedad de infraestructuras a través de un enfoque guiado y práctico.
